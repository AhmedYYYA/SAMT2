# Design System

## Direction

The new design is an editorial digital-art experience rather than a conventional corporate landing page. It preserves the seriousness of an executive aviation fellowship while using controlled cinematic motion.

## Core palette

- Background: `#050507`
- Secondary background: `#08080d`
- Primary text: `#f5f5f3`
- Muted silver: `#8c8c98`
- Violet accent: `#8f64ff`
- Cyan accent: `#5dd7ff`

## Typography

- English display: Syne
- English body: Inter
- Arabic: Noto Sans Arabic

## Motion principles

1. Motion clarifies hierarchy and sequence.
2. No rapid or decorative-only animation.
3. Horizontal movement is reserved for the leadership journey.
4. International stations change through cross-fade and controlled vertical movement.
5. Reduced-motion users receive a static, readable experience.
